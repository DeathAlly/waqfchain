System Architecture · v1.0

WaqfChain Technical Blueprint

End-to-end architecture for tokenized Islamic endowment assets on Arc · Powered by Circle infrastructure

Layer 1 — User Interface
🖥 Frontend
WaqfChain Web App
React/HTML · Tokenize, distribute, monitor compliance
Built with vanilla JS + HTML. Connects to MetaMask via window.ethereum. Routes all transactions through Arc RPC. Hosted on Vercel.
📱 Wallet
MetaMask / Circle Wallet
User wallet · Signs txns · Arc testnet configured
Users connect MetaMask configured with Arc testnet RPC. Circle Wallets SDK used for beneficiary wallet management and KYC linking.
🔐 KYC
Circle Identity / KYC
Investor accreditation · Beneficiary verification
Circle's KYC/identity layer verifies investors and beneficiaries. Verification status is stored onchain as a flag in the compliance contract.
API Calls / wallet_signTransaction
Layer 2 — Circle Infrastructure
💵 Circle
USDC
Native settlement currency · Dollar-denominated fees · Distribution payouts
All yield distributions and token settlements are denominated in USDC. Arc uses USDC as the native gas token, giving predictable, dollar-denominated fees.
🏦 Circle
Circle Wallets
Programmable wallets · Beneficiary key management · Multi-sig treasury
Circle Wallets SDK manages beneficiary wallets programmatically. The Waqf treasury uses a multi-sig Circle Wallet. Agent-initiated payments use wallet policies for authorization.
🔀 Circle
Circle Gateway
Treasury routing · Payout orchestration · Liquidity management
Circle Gateway handles treasury liquidity and routes outgoing USDC distributions to beneficiary wallets. Used for multi-party settlement flows.
🌉 Circle
CCTP + Bridge Kit
Cross-chain USDC transfers · GCC multi-jurisdiction settlement
CCTP (Cross-Chain Transfer Protocol) enables USDC movement between Arc and other chains. Used when beneficiaries in different jurisdictions need cross-chain settlement.
eth_sendRawTransaction · Arc RPC
Layer 3 — Arc Blockchain (Circle L1)
📜 Onchain
WaqfToken.sol
ERC-20 · Fractional ownership · Transfer restrictions · Compliance flag
Core ERC-20 token contract with transfer restrictions enforced onchain. Only KYC-verified addresses can hold or receive tokens. Pausing and freezing controlled by the compliance contract.
⚖️ Onchain
ComplianceRegistry.sol
KYC flags · Sharia compliance · AWQAF registration hash · Investor whitelist
Stores compliance state for each asset and investor. KYC status, Sharia screening results, and AWQAF authority registration hashes are written here. Called by WaqfToken on every transfer.
💸 Onchain
DistributionEngine.sol
Yield calculation · Pro-rata USDC payouts · Beneficiary splits · Zakat reserve
Calculates yield distributions proportionally to token holdings. Enforces beneficiary split ratios (e.g. 40% education, 35% healthcare). Automatically reserves 2.5% Zakat. Distributes USDC via Circle Gateway.
🗳 Onchain
WaqfGovernance.sol
Mutawalli (trustee) multisig · Parameter changes · Emergency pause
Governance contract where registered Waqf trustees (Mutawallis) vote on parameter changes, asset updates, and emergency pauses. Requires 2-of-3 trustee signatures for critical actions.
Events · eth_getLogs · Indexing
Layer 4 — Data & Compliance Oracle
🗄 Oracle
AWQAF Authority Feed
Off-chain Waqf registry · Registration verification · Asset legitimacy
Off-chain oracle reads from UAE AWQAF authority registration database. Verifies Waqf registration numbers and writes confirmation hashes to ComplianceRegistry.sol.
📊 Indexer
Event Indexer
Arc event logs · Transaction history · Distribution records · Audit trail
Indexes all onchain events (TokenIssued, DistributionExecuted, ComplianceUpdated) into a queryable database. Powers the dashboard's transaction log and audit trail views.
🏘 Storage
Asset Metadata (IPFS)
Property docs · Valuation reports · Legal certificates · Sharia fatwa
Asset documentation (title deeds, valuation reports, Sharia fatwa certificates) stored on IPFS. Content hashes pinned to the token metadata onchain for immutable reference.
End-to-End Transaction Flow
01
Asset Registration
Mutawalli submits Waqf asset details + AWQAF reg. number via frontend
02
Compliance Check
ComplianceRegistry validates KYC, Sharia screening, AWQAF oracle confirmation
03
Token Issuance
WaqfToken.sol mints ERC-20 tokens on Arc. Metadata hash pinned to IPFS
04
Yield Accrual
DistributionEngine calculates pro-rata yield, reserves Zakat, queues USDC payouts
05
USDC Settlement
Circle Gateway routes USDC to verified beneficiary wallets. Sub-second finality on Arc
